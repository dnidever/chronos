Installing chronos
=====================

Until I get chronos set up with PyPi, it's best to install it from source from GitHub.

.. code-block:: bash

    pip install git+https://github.com/dnidever/chronos


Dependencies
============

- numpy
- scipy
- astropy
- matplotlib
- `dlnpyutils <https://github.com/dnidever/dlnpyutils>`_
